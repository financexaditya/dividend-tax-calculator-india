## About The Project
Calculating dividend tax in India can be confusing due to shifting tax slabs, TDS (Tax Deducted at Source) rules under Section 194, and surcharge calculations. This repository outlines the core logic and variables used to build a streamlined dividend tax calculator for retail investors.

We built this tool to help investors easily project their actual take-home dividend yield after the government takes its cut.

## Core Features
* **Automated TDS Deduction:** Factors in the standard 10% TDS if total dividends exceed ₹5,000 in a financial year.
* **Marginal Tax Rate Integration:** Calculates final liability based on the user's specific income tax bracket.
* **Surcharge & Cess:** Includes applicable health and education cess logic.
* **Public Portfolio Tracking:** Built alongside India's first fully transparent public dividend portfolio.
